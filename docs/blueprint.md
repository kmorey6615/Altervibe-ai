# **App Name**: AlterVibe AI

## Core Features:

- User Authentication & Profiles: Secure sign-up and login via Firebase Authentication, with each user maintaining a personalized profile.
- AI Character Builder: Users can create and save unique AI influencer characters by defining attributes like name, age, style, and personality. The tool guides the setup of these AI persona definitions, stored in Firestore.
- Content Studio Selection: Interface for users to select an existing AI character and a desired content style (e.g., viral dance, aesthetic, POV, lip sync) as the basis for new short video content.
- AI Caption & Placeholder Video Generation: The AI tool generates a contextual, creative caption relevant to the chosen character and content style. A placeholder video is displayed, simulating the final output.
- Dynamic Content Feed: A scrollable feed, similar to TikTok, showcasing all generated videos with their respective usernames, captions, and engagement icons.
- Video Export & Caption Utility: Functionality allowing users to download their generated placeholder videos and a convenient 'Copy Caption' button to easily share the AI-generated text.
- User Performance Dashboard: A dashboard displaying core user statistics, such as the total number of videos created, along with placeholders for future metrics like views and followers.

## Style Guidelines:

- Primary Background: Deep black (#000000) for a sleek, modern, and high-contrast foundation.
- Secondary Background/Surfaces: Dark grey (#1A1A1A) for subtle visual separation and depth.
- Primary Text/Foreground: Crisp white (#FFFFFF) for maximum readability against dark backgrounds.
- Secondary Text/Inactive Elements: Light grey (#CCCCCC) for supporting information and disabled states.
- Accent Color 1 (Primary Interaction): Vibrant purple (#8A2BE2) for primary interactive elements, highlights, and CTAs.
- Accent Color 2 (Secondary Interaction/Highlights): Energetic yellow (#FFD700) for secondary interactive elements, notifications, and special emphasis.
- Body and Headline Font: 'Inter' (sans-serif) for its modern, clean, and highly readable qualities, perfectly complementing a minimal UI on both mobile and desktop screens.
- Utilize a consistent set of minimalist line-based icons for navigation, engagement actions, and UI elements, reinforcing the clean, modern, and uncluttered design ethos.
- Adopt a mobile-first, clean layout with a 'TikTok-style' vertical scrolling feed, clear hierarchy, smooth button interactions, and a simplified navigation bar (Home, Create, Profile) for intuitive user experience.
- Incorporate subtle, smooth animations for feed scrolling transitions, button hovers, and other interactive elements to enhance the user experience without distracting from the minimalist aesthetic.
